This script is a simple links shortener. The user enters a link that should be cut and gets a randomly generated short link. Entered links are checked for correctness and validation. In addition, you can choose your own version of short links, entered by the user short links are checked for duplication. Information about compliance of long and short links entered in the database. The same user can share a short link to the other. Script settings in a separate file settings.php (configure the database, the length of the short links).
